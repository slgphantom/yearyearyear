安裝eclipse 及 jdk完畢后:
打開 eclipse

1. 創建一個工作空間
2. 進入主界面
3. 創建java project
4. 關閉歡迎頁面
5. 把歡迎頁面關閉即可進入如下界面
6. 右鍵主文件 準備添加pgo api及 tg api
7. 導入兩個api文件
8. 開始寫你的第一個java程序
9. 打開你所創建的java文件 并輸入指令
10. 配置程序入口
11. 配置完成并執行
12. 如果出現hello world 恭喜你 java配置成功


指令:

public classExample{
    public static void main(String args[]){
        System.out.println("Hello World!");
    }
}
